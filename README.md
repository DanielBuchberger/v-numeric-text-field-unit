# v-unit-numeric-text-field

Vue 3 component for numeric text fields with unit conversion.

## Installation

```bash
npm install v-unit-numeric-text-field
```

## Usage

```vue
<script setup>
import { VNumericTextField, VNumericTextFieldUnit } from 'v-unit-numeric-text-field'
</script>

<template>
  <VNumericTextField />
  <VNumericTextFieldUnit />
</template>
```
